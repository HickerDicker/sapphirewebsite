FAQ

Does SapphireOS support X?

Some services/components are disabled by default. If you face issues, try enabling the standard services. If it still fails, that feature may simply be unsupported.

Why SapphireOS?

SapphireOS is “source available” rather than fully open source. The NTlite XMLs are the only non available component.

Why isn’t X driver working on Windows Server?

Windows Server 2022 differs from standard Windows versions, which can cause compatibility issues with some drivers, especially those from AMD and certain Intel NICs.

Superlite vs Ultralite?

Ultralite removes more components than Superlite for a smaller ISO and slightly faster boot times. The core tweaks remain the same.

SapphireOS

SapphireOS is a modified Windows ISO designed for gaming. It offers a curated set of tweaks and performance optimizations alongside powerful post-install utilities.
